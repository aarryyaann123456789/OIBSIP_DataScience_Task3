# Car Price Prediction
**Task 3 for Oasis Infobyte Internship**

## Objective
To predict the price of used cars based on features like Year, Selling Price, KM Driven, Fuel Type, etc.

## Tools Used
* Python
* Pandas (Data manipulation)
* Scikit-Learn (Machine Learning)
* Matplotlib (Visualization)

## Outcome
Successfully trained a Linear Regression model that predicts car prices with reasonable accuracy based on the provided dataset.
